# AGENT_PROTOCOL.md

## Note
All keywords and example prompts currently work correctly only with the component tests and the script shortcuts defined in .ai/testing/package.json.

## Agent Protocol: Keywords & Example Prompts

### `generate`
**Purpose:** Create new files, components, or tests.

**Prompt Examples:**
- generate components for the shadow property of the @Component decorator

### `run`
**Purpose:** Execute scripts, tests, or coverage checks.

**Prompt Examples:**
- run coverage
- run cmp tests
- run tests

### `build`
**Purpose:** Build components, packages, or the project.

**Prompt Examples:**
- build cmp
